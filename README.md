# Brazilian E-Commerce Dashboard

## Project Overview
This project presents an interactive Power BI dashboard using the Brazilian E-Commerce (Olist) dataset from Kaggle.

## Tools Used
- Power BI Desktop
- Microsoft PowerPoint
- GitHub

## Dataset
Brazilian E-Commerce Public Dataset by Olist (Kaggle)

## Dashboard Features
- Total Revenue KPI
- Total Orders KPI
- Total Customers KPI
- Average Order Value KPI
- Sales by Month
- Revenue by State
- Revenue by Product Category
- Payment Type Distribution
- Interactive Slicers
